# quiz-game

🧠 Quiz Game App
A modern and interactive quiz game designed to test knowledge, improve learning, and provide an engaging user experience. This project showcases front-end development skills, logical problem-solving, and clean UI design.

🚀 Features
🎯 Multiple-choice questions
⏱️ Real-time scoring system
🔄 Instant feedback on answers
📊 Final score display
🎨 Clean and responsive UI
⚡ Fast and smooth gameplay

🛠️ Tech Stack
Frontend: HTML, CSS, JavaScript
Version Control: Git & GitHub

📂 Project Structure
quiz-game/
│── index.html
│── style.css
│── script.js
└── README.md

⚙️ Installation & Setup
Clone the repository
Bash
git clone https://github.com/utazifredrick/quiz-game.git

Navigate into the project folder
Bash
cd quiz-game
Open in browser
Double-click index.html
OR
Use Live Server (recommended)

🧩 How It Works
The game presents a series of questions
Users select an answer from multiple choices
The app checks if the answer is correct
Scores are updated in real time
Final score is displayed at the end
